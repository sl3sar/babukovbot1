# Telegram Task Planner Bot

Простой планировщик задач для Telegram с календарем и временем выполнения.

## Возможности

- Добавление задач с датой/годом/временем через календарь
- Опциональное описание задачи
- Авто-напоминания о задачах (за 3 дня и за 1 день)
- Еженедельный обзор по понедельникам
- Просмотр активных задач
- Просмотр всех задач (включая выполненные)
- Отметка задачи как выполненной
- Удаление задачи
- Очистка выполненных задач

## Команды бота

- `/start` - приветствие и помощь
- `/help` - список команд
- `/add <название>` - добавить задачу (название → описание → дата → время)
- `/skip` - пропустить ввод описания при добавлении
- `/calendar` - открыть календарь
- `/list` - активные задачи
- `/all` - все задачи
- `/done <id>` - отметить задачу выполненной
- `/delete <id>` - удалить задачу
- `/clear_done` - удалить все выполненные задачи

## Установка и запуск

1. Установи Python 3.10+.
2. Установи зависимости:
   `py -m pip install -r requirements.txt`
3. Скопируй `.env.example` в `.env` и укажи `BOT_TOKEN`.
4. Запусти бота:
   `py main.py`

## Как загрузить в Telegram

1. Открой [@BotFather](https://t.me/BotFather).
2. Выполни `/newbot` и создай бота.
3. Скопируй выданный токен.
4. Подставь токен в `BOT_TOKEN`.
5. Запусти `py main.py`.
6. Открой своего бота в Telegram и отправь `/start`.

## Как добавляется задача с датой и временем

1. Отправь команду `/add Встреча с клиентом`.
2. Отправь описание одним сообщением (например: `созвон + подготовить вопросы`) или отправь `/skip`.
3. В календаре нажми нужный день (в нужном месяце и году).
4. Отправь время в формате `ЧЧ:ММ`, например `18:45`.
5. Бот сохранит задачу с точной датой и временем.

## Уведомления

- Напоминания отправляются каждый день примерно в **09:00** (по времени сервера), если есть задачи:
  - за **3 дня**
  - за **1 день (завтра)**
- По понедельникам примерно в **09:05** бот присылает список задач на неделю (по дням).
